Application Name: AI-Powered Collaborative Dinner Planner

Description: The AI-Powered Collaborative Dinner Planner is an interactive web application designed to help groups collaboratively plan and prepare a dinner party. The platform combines real-time collaboration with artificial intelligence to transform available ingredients into curated dish suggestions, enabling teams to make collective, informed decisions.

Overview

This application is built for groups who want to organize a dinner but may not have predefined recipes or cooking plans. By leveraging AI-driven recommendations and a structured voting system, users can efficiently select the perfect dishes for their event.

Tech Stack:

- Frontend: React.js (Vite) + Tailwind CSS (for styling) + Socket.io Client.

- Backend: Node.js + Express.js + Socket.io Server.

- Database: MongoDB (Mongoose ODM).

- Authentication: JWT (JSON Web Tokens).

- AI/LLM (Free & Best): Google Gemini API (Gemini 1.5 Flash).
